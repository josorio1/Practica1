2.Dataset de Kaggle: https://www.kaggle.com/datasets/gpreda/reddit-wallstreetsbets-posts
3.Paquete que usan para realizar el dataset: https://praw.readthedocs.io/en/latest/#getting-started
4.Es un wrapper de la API de Reddit para Python, así que ya tenemos una motivación: replicar la estructura utilizando web scraping
en vez de la API.
5.Otra cosa a vender es que el dataset real no tiene los comentarios de los posts, y es algo que se podría añadir. Esto sirve para cubrir lo de "descubrimiento de enlaces y navegación autónoma".
